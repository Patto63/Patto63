## 📌 Proyectos destacados

### 🏥 Medicity — Sistema Hospitalario Monorepo
Arquitectura de microservicios con API Gateway, servicios de autenticación y administración, módulo de consultas por región, aplicaciones web (admin y centros) y documentación centralizada.

**Tecnologías:** Node.js, NestJS, React, PostgreSQL, Docker, Azure DevOps, CI/CD.

### ⛽ Gestión de Combustibles
Sistema empresarial para administración de vehículos, combustibles, usuarios y transacciones. Basado en microservicios con gRPC y API Gateway.

**Tecnologías:** ASP.NET Core, SQL Server, Docker, gRPC, GitHub Actions.

### 🔐 Sistema de Archivos Seguros
Aplicación para almacenamiento y gestión de archivos cifrados, control de permisos, firmas digitales y auditoría de accesos.

**Tecnologías:** Node.js, React, MariaDB, AES, JWT.

### 🤖 Reconocimiento Facial
Herramienta de autenticación biométrica utilizando modelos de reconocimiento facial y visión por computador.

**Tecnologías:** Python, OpenCV, Machine Learning.

### 📊 Data Warehouse y Business Intelligence
Implementación de procesos ETL y modelo dimensional para visualización y análisis de datos en un data warehouse orientado a la toma de decisiones.

**Tecnologías:** SQL Server, Python, PySpark, Power BI.

### 🧪 gitt-api — API de laboratorio CI/CD
Repositorio de ejemplo que muestra cómo integrar CI/CD con GitHub Actions y despliegue automatizado.
